# 北京南站附近的酒店及路线

## 米家青年酒店(北京南站店)
- **地址**: 右外东庄21号楼
- **步行距离**: 304米
- **步行时间**: 约4分钟
- **路线详情**:
  1. 沿南站幸福路向西步行98米右转
  2. 沿南站幸福路向东步行66米左转
  3. 沿幸福四巷向北步行140米到达目的地

## 汉庭酒店(北京南站护城河店)
- **地址**: 开阳路6号院1号楼
- **步行距离**: 851米
- **步行时间**: 约11分钟
- **路线详情**:
  1. 沿南站幸福路向西步行98米向右前方行走
  2. 沿南站幸福路向西步行212米右转
  3. 沿开阳路向北步行280米右转
  4. 沿右安门东滨河路向东步行161米右转
  5. 向南步行44米左转
  6. 向东步行11米向右前方行走
  7. 向南步行26米左转
  8. 向东步行19米到达目的地
